Lengkapi halaman utama Anda dengan plugin pencarian penerbangan pesawat Whitelabel Anda

== Description ==

Dengan menggunakan Plugin ini, Anda dapat memiliki form pencarian Penerbangan yang mengarah langsung ke halaman Whitelabel Anda

Fitur yang dapat Anda nikmati:

*   Menggunakan APTMI Trip Widget pada sidebar Anda
*   Menggunakan shortcode [aptmi_search] pada satu halaman penuh Anda.  

== Installation ==

Sangat mudah untuk memasang plugin ini,

1. Untuk menginstal plugin caranya cukup mudah. Apabila Anda ingin mengupload langsung menggunakan file zip. Melalui wordpress pada menu "Plugins" > "Add New" > "Upload Plugin"
1. Aktifkan Plugin melalui layar 'Plugins' dalam admin WordPress
1. Maka Widget "APTMI Widget" dapat Anda lihat di menu "Appearance" > "Widget"
1. Anda juga dapat menggunakan shortcode [aptmi_search] jika Anda ingin menggunakannya sebagai satu halaman penuh


== Frequently Asked Questions ==

= Versi WordPress saya sangat lama, apakah saya bisa menggunakannya =

Ya, pada dasarnya semua versi Wordpress dapat menjalankan Plugin ini

= Apa saja yang saya dapat atur =

Pada widget, Anda dapat mengatur "height" atau tinggi widget dalam pixel agar form pencarian pesawat dapat terlihat semua

Silakan untuk tidak menambahkan akhiran "px" atau "%" dalam kolom isian, karena Plugin akan otomatis menambahkannya dalam satuan "px"

Perlu diperhatikan bahwa isian "URL B2C" adalah wajib

Sama seperti shortcode, Anda wajib mengatur "b2c_url" seperti berikut: 
>[aptmi_search b2c_url="http://domainanda.com/" ]

width dan height juga dapat Anda masukkan dalam shortcode ini, misal:

>[aptmi_search b2c_url="http://domainanda.com/" width="800"]
 .
